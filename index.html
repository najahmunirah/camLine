InFrame Synapse MES

Manufacturing Execution System for high-tech fabrication sites

With its open architecture, InFrame Synapse MES (Manufacturing Execution System) supports the communication capability of manufacturing with discrete processes. It provides rich functionalities combined with easy usability in the area of production planning, monitoring, and control for high-tech fabrication sites.

 Service oriented architecture (SOA) to enable fast integration with other IT system

 Process oriented organization

 Rapid integration

 Direct entry into Tracking & Tracing

InFrame Synapse Equipment Connector

Powerful interface for easy to use equipment integration

The InFrame Synapse Equipment Connector allows an OEM to add a data interface to nearly any equipment PLC, with minimal effort! Well known examples of such data interfaces are SEMI SECS/GEM, SEMI PV02 or CONTINENTAL GHP. Either way no detailed knowledge on the underlying protocol or its architecture is required for the PLC programmer. The configuration and documentation are easily done with a single Excel spreadsheet: PLC function blocks, configuration, and documentation are created by simply clicking on a button.

Our solutions:

 Compliance with SEMI SECS / GEM 200mm and 300mm standards, as well as SEMI-PV02

 Highly flexible process data recording

 All major PLC products
 PCs with industrial applications

 Optional MES interfaces
